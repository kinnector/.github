# .github

This repository contains organization-wide configurations and profile assets for the `kinnector` GitHub organization.

## Contents

- **`profile/`**: Contains the organization profile README (`profile/README.md`) displayed on the `kinnector` organization page.

## Adding Organization Defaults

To define organization-wide default templates, guidelines, or community health files, add them to this repository following standard GitHub paths (e.g., `.github/ISSUE_TEMPLATE/` or `.github/workflows/`).
